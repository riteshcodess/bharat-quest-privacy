# Bharat Quest — Official Privacy Policy Website

Modern, responsive single-page Privacy Policy website designed specifically for the **Bharat Quest** Android mobile game on the Google Play Store.

## Live URL
Once pushed to GitHub and enabled under repository **Settings &rarr; Pages**, the site is accessible at:
👉 **https://riteshcodess.github.io/bharat-quest-privacy/**

---

## Features
- **Pure HTML/CSS/Vanilla JS:** Zero build steps, zero npm dependencies, instant loading.
- **Google Material & Bharat Quest Gold Theme:** Deep blue background (`#08111F`), glassmorphic rounded cards, glowing gold accents (`#F6B73C`), and ambient background orbs.
- **Google Play Store Compliant:** Explicit sections detailing local device storage, AdMob advertising identifiers (`AD_ID`), and in-app data deletion controls ("Reset Progress").
- **Mobile-First & Responsive:** Smooth navigation bar, clean typography (`Cinzel` & `Inter`), and optimized touch targets.

---

## Files Structure
- `index.html` — The single-page privacy policy containing all legal disclosures, structured sections, and navigation.
- `styles.css` — Pure CSS tokens, glassmorphic card layouts, responsive breakpoints, and animations.
- `README.md` — Documentation and usage guide.

---

## How to Customize & Deploy
1. **Update Email Address:**
   If you wish to change the support contact email (`support@bharatquest.app`), simply open `index.html` and search for `support@bharatquest.app`, then replace it with your personal or developer email.
2. **Push to GitHub:**
   ```bash
   git add .
   git commit -m "Deploy production-ready Bharat Quest Privacy Policy site"
   git push -u origin main
   ```
3. **Google Play Console Link:**
   In your Google Play Console dashboard under **App Content &rarr; Privacy Policy**, paste your live URL:
   `https://riteshcodess.github.io/bharat-quest-privacy/`
